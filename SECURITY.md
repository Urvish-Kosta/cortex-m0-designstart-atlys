# Security policy

This is an educational FPGA project with no network surface, no secrets and no
production deployment. Still:

- **Reporting**: open a GitHub issue, or for anything sensitive email
  kostaurvish@gmail.com.
- **Scope**: plausible issues are limited to things like unsafe firmware
  memory handling or CI supply-chain concerns (vendored Unity is pinned to
  v2.6.0 and checked in, not fetched at build time).
- **Arm IP**: if you find Arm-licensed material accidentally committed to the
  history, report it immediately so it can be purged.
